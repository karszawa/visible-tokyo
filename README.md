## Local development

### Requirements

```
$ node -v
v8.1.2
$ yarn -v
1.3.2
```

### Build

```
$ git clone --depth 1 git@github.com:InfovisHandsOn/A_make-tokyo-great-again.git
$ cd make-tokyo-great-again
$ yarn
$ npm start # visit localhost:3000
```

## Features

- 住宅探しの観点から東京都の各地域の特色を可視化する
- ひとめで地域ごとの特色がわかる地図を表示する
- 上京してきた大学生の引越しに役立てたい
